First step is to add canonical tags to the blog. I modified layouts/partials/head.html and added a canonical tag just above the css link tags. I tried to copy the code itself here, but I keep running into issues with Hugo not wanting to show it. So, you’ll get a picture instead.

Next step is to sign-up for Hyvor Talk. Choose “Manual Installation”. Copy the code/script that they provide and paste it inside layouts/post/single.html just before the last {{ end }} at the bottom of the file.

And that’s it.
